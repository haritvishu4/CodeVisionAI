# 🚀 CodeVisionAI

A professional Python project analyzer that scans Python projects and generates useful insights.

## ✨ Features

- 📁 Project Scanner
- 🐍 Python File Counter
- 📊 Lines of Code Counter
- 🔧 Function Counter
- 🏛️ Class Counter
- 📦 Import Analyzer
- 📝 TODO / FIXME Detector
- 🔐 Secret Detector
- 📄 HTML Report Generator

## 📂 Project Structure

```text
CodeVisionAI/
├── src/
│   ├── main.py
│   ├── scanner.py
│   ├── analyzer.py
│   └── report.py
├── reports/
├── README.md

└── .gitignore
```

## ▶️ Installation

```bash
git clone https://github.com/haritvishu4/CodeVisionAI.git
cd CodeVisionAI
pip install -r requirements.txt
python3 src/main.py
```
